Help

Starting a Game

Open Options, choose a difficulty, then select New Game. The puzzle will begin immediately and your progress will save automatically.

Difficulty Levels

Easy allows up to 7 mistakes. Medium allows up to 5 mistakes. Hard allows up to 3 mistakes and disables the Hint feature. Turning Clock or Mistakes off changes the displayed difficulty to Custom.

Playing the Puzzle

Select an empty square, then choose a number from the number buttons. You can also use your keyboard number keys.

Notes Mode

Use Notes Mode when you want to mark possible answers in a square before choosing the final number.

Clock and Mistakes

The Clock and Mistakes options can be enabled or disabled independently. When either option is turned off, the game is considered a Custom game. Clock Off disables the timer. Mistakes Off removes the mistake limit.

Erase, Hint, and Pause

Erase clears the selected square. Hint fills one correct answer and is disabled on Hard difficulty. Pause hides the board and pauses the timer until you resume play.

Game Over

If Mistakes are enabled and you reach the maximum mistake limit for the current difficulty, the game ends immediately. The board becomes read-only, the timer stops, and the game is recorded as a loss.

Automatic Saving

The game saves automatically as you play. If you refresh the browser, the puzzle should reopen where you left off.

Statistics

The Statistics window tracks games played, wins, losses, fastest times, streaks, hints used, mistakes made, valid numbers played, erases used, and other gameplay records.

Import and Export Backups

Use the Admin window to export a backup file or import a backup file. Backup imports must match the current game version to protect your data.
